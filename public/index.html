后期云端化建议

当前原型使用 JSON 文件和本地附件目录持久化，接口边界已经按云端系统设计。正式部署时建议替换为：

  • 数据库：PostgreSQL / MySQL，记录量可扩展到百万级。
  • 附件：对象存储 OSS / S3 / MinIO，支持权限和生命周期。
  • 权限：企业微信/飞书/OIDC 登录，角色分为管理员、设计、工艺、质量、只读。
  • 流程：新增、评审、受控、变更、作废均进入审批流。
  • 集成：BOM、ERP、PLM、NAS 文件归档通过 API 对接。
